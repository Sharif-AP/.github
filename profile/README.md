# Advanced Programming (AP) — Sharif University of Technology

Welcome to the official GitHub organization of the **Advanced Programming (AP)** course at **Sharif University of Technology**.  
This space contains course materials, assignments, and real-world projects designed to help students master software engineering and object-oriented programming.

---

## Course Overview
The **Advanced Programming** course focuses on building strong programming foundations through:
- Object-Oriented Design and Programming (OOP, UML, Design Patterns)  
- Software development lifecycle and architecture design  
- Hands-on team projects and version control using Git & GitHub  
- Working with Java and Python in modern development environments  
- Applying principles of clean code, modularity, and scalability

---

| Repository | Description |
